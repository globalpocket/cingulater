# Agent Behavior Policy (Antigravity)

このリポジトリで作業する AI エージェント（特に関数・ツールを使用するエージェント）が必ず遵守すべき鉄則を定義します。

## 1. Browser Tool Usage
- **クローズ義務**: ブラウザ操作（`browser_subagent` 等）を使用した場合、その指示の最後には必ず「セッションを正常に終了し、クローズすること」という旨を含めなければならない。
- **報告義務**: ブラウザツールからの最終レポートに「クローズ完了」が含まれていることを確認するまで、そのターンを完了してはならない。

## 2. Process Hygiene
- **残留プロセスの禁止**: `headless_shell`, `Chromium`, `chrome`, `node` などのプロセスを背後に残したまま「作業完了」を報告してはならない。
- **終了時チェック**: 全てのタスク完了時に、`.agent/scripts/hygiene_check.py` を実行し、環境が清浄であることを確認しなければならない。

## 3. Mandatory Cleanup
- **異常終了時の対応**: 何らかの理由でブラウザセッションがスタックした場合、直ちに `/cleanup` ワークフローを実行し、環境をリセットしなければならない。

---
*この規約はリポジトリの整合性と開発環境の健全性を維持するためのものであり、いかなる場合も無視してはならない。*
